# ryrun Home Assistant Add-ons

Home Assistant add-on repository maintained by ryrun.

## Add-ons

- `ctrlapp`: CtrlApp UI served as a Home Assistant add-on.
- `unbound`: Unbound DNS resolver add-on imported from `fenio/ha-addons`.
